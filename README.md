# cmajorMiniProgram
微信移动终端开发课程设计-转专业信息登记系统后端
## 前端

1. 注册登录实现、前后端数据交互
2. 学生端
   - 主页
   - 转专业申请填写、附件上传
   - 查看已提交申请
   - 修改已提交申请
   
   
3. 教师端
   - 主页
   - 查看所有转专业申请（列表形式）
   - 查看单个申请（最后点击是否通过申请）
   - 查看统计信息
   - 导出数据

## 后端
接口包括：

1. 登录
3. 获取已填写申请
4. 申请信息表单提交处理
5. 文件上传接口
6. 文件下载接口
7. 获取导出压缩包
8. 获取全部申请
9. 获取统计信息
