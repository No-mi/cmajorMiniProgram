# cmajorMiniProgram
微信移动终端开发课程设计-转专业信息登记系统后端
## 前端

1. 注册登录实现、前后端数据交互
2. 学生端
   - 主页（包括登录/注册、个人信息、当前状态、修改/填写转专业申请（根据当前状态））
   - 转专业信息填写、附件上传
   - 通知公告列表（点击公告查看详情）
   - 单个公告查看
3. 教师端
   - 主页（登录/注册)
   - 查看所有转专业申请（列表形式）
   - 查看单个申请（最后点击是否通过申请）
   - 通知公告列表（点击公告查看详情，添加公告按钮）
   - 单个公告编辑（包含删除按钮）

## 后端



接口包括：

1. 登录
2. 个人信息获取
3. 申请信息获取
4. 申请信息表单提交处理
5. 公告列表获取
6. 单个公告获取
7. 教师端添加公告提交处理
8. 公告删除
9. 学生申请列表获取
